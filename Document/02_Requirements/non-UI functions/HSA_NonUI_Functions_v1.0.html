📋 Rang buoc Phi Chuc nang (NFR) lien quan den Non-UI Functions

⚡ Hieu suat Auto-grading
API nhan bai thi va tra ve Score + Ket qua phai hoan thanh trong < 1,000ms ke tu thoi diem nhan request (ke ca khi co 2,000 HV nop bai dong thoi).
📧 Email SLA
Enrollment Confirmation Email phai duoc gui trong vong 30 giay sau khi NV click Confirm. OTP Email phai den trong < 60 giay.
🔸 Offline Resilience
Answer Cache Service phai luu du lieu cau tra loi tuc thoi vao LocalStorage. Khi ket noi phuc hoi, Sync Service phai dong bo trong < 5 giay ma khong mat du lieu.
🔒 Bao mat Token
Parent Report Token phai co Time-to-Live (TTL) 72 gio. JWT Access Token het han sau 8 gio. Refresh Token: 30 ngay. Tokens phai duoc ky bang HMAC-SHA256.
📊 Adaptive Engine Rule
Competency Profile chi duoc cap nhat khi Answered Rate (so cau da tra loi / tong so cau) > 30%. Bai tin hieu hoac bo cuoc khong tinh.
📋 Audit & Compliance
Moi hanh dong write (Confirm Enrollment, Approve QB, Delete User) phai ghi Audit Log voi: Actor ID, Timestamp (UTC), Action type, Before/After state. Log luu tru toi thieu 1 nam.
